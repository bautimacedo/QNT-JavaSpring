# Agente Tester — Redirección

El **Agente Tester** está definido en:

**Importante-main/AgenteQA/AGENTE_TESTER.md**

Para ejecutar la verificación, usá ese archivo:

```
Lee Importante-main/AgenteQA/AGENTE_TESTER.md y ejecutá la verificación.
```

Ese agente ejecuta build + tests JUnit (y opcionalmente smoke API) y emite el reporte estándar. Forma parte del pipeline secuencial PM → Programador → Tester; ver **Importante-main/PIPELINE_SECUENCIAL.md**.
