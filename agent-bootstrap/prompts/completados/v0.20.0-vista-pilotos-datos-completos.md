**PRIORIDAD:** alta

## Descripción

La vista "Pilotos" del frontend necesita mostrar, por cada piloto, sus datos operativos: nombre, apellido, email, horas de vuelo, cantidad de vuelos, vencimiento del CMA, estado de cuenta, y un resumen de sus licencias ANAC.

Actualmente, `GET /usuarios/pilotos` devuelve la lista de entidades `Usuario` completas, que incluyen campos innecesarios y podrían exponer datos sensibles. Además no incluye las licencias ANAC de cada piloto.

Esta tarea crea un DTO `PilotoResumenResponse` y actualiza el endpoint `GET /usuarios/pilotos` para devolver ese DTO, con toda la info que el frontend necesita para la vista de pilotos.

Reemplazar el método `listPilotos()` para que devuelva el DTO:

```java
@GetMapping("/pilotos")
@Transactional(readOnly = true)
@PreAuthorize("hasRole('ADMIN')")
public ResponseEntity<List<PilotoResumenResponse>> listPilotos() {
    try {
        List<Usuario> pilotos = usuarioBusiness.listPilotos();
        List<PilotoResumenResponse> response = pilotos.stream().map(p -> {
            List<LicenciaANAC> licencias;
            try {
                licencias = licenciaANACBusiness.listByPiloto(p.getId());
            } catch (BusinessException e) {
                licencias = List.of();
            }
            List<PilotoResumenResponse.LicenciaANACResumen> licenciasDto = licencias.stream()
                .map(l -> new PilotoResumenResponse.LicenciaANACResumen(
                    l.getId(),
                    l.getFechaVencimientoCma(),
                    l.getFechaEmision(),
                    l.getCaducidad(),
                    l.getImagenCma() != null && l.getImagenCma().length > 0,
                    l.getImagenCertificadoIdoneidad() != null && l.getImagenCertificadoIdoneidad().length > 0,
                    l.getActivo()
                ))
                .toList();

            return new PilotoResumenResponse(
                p.getId(),
                p.getNombre(),
                p.getApellido(),
                p.getEmail(),
                p.getHorasVuelo(),
                p.getCantidadVuelos(),
                p.getCmaVencimiento(),
                p.getEstado(),
                p.getImagenPerfil() != null && p.getImagenPerfil().length > 0,
                licenciasDto
            );
        }).toList();

        return ResponseEntity.ok(response);
    } catch (BusinessException e) {
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).build();
    }
}
```

**Verificación:** `GET /api/qnt/v1/usuarios/pilotos` con token ADMIN devuelve un array de objetos con los campos del DTO, incluyendo las licencias de cada piloto.
